# Konsensus Flow - Diagram Editor

Web-based editor for Mermaid and PlantUML diagrams.

## Features
- Dual diagram engine support
- Project/folder organization
- Template library
- Real-time preview
- Export as SVG/PNG

## Tech Stack
- Next.js
- TypeScript
- Monaco Editor
- Mermaid.js
- PlantUML
- Prisma ORM

## Quick Start
```bash
npm install
npm run dev
```

## API
- `/api/projects/[id]/diagrams` - Diagram operations
- `/api/projects/[id]/folders` - Folder management
